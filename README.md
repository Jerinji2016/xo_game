# xo_game

A Flutter app to play Tic Tac Toe with Neomorphic UI

## Getting Started

1. Clone the repository  

    ```sh
    git clone git@github.com:Jerinji2016/xo_game.git
    ```  

2. Get dependencies and run

    ```sh
    flutter pub get && flutter run
    ```
